# ♻️ WhereIsBin

WhereIsBin is an Android-based smart waste reporting application designed to help citizens report illegal dumping and locate proper disposal facilities. The app connects users and administrators through a real-time Firebase backend to promote cleaner and more responsible communities.

---

## 🚀 Features

### 👤 User Features
- 🔐 Secure login & registration (Firebase Authentication)
- 📝 Report illegal waste dumping
- 📍 Submit location-based complaints
- 📷 Upload image evidence
- 📊 Track report status (Pending / Approved / Rejected)
- ♻️ Find legal disposal centers (Dustbin, E-Waste, Scrap Stores)

### 🛠️ Admin Features
- 📋 View all submitted reports
- ✅ Approve reports
- ❌ Reject reports
- 📌 View complaint details including location
- 📊 Manage categorized reports (Pending / Approved / Rejected)

---

## 🏗️ Tech Stack

- **Language:** Java
- **Platform:** Android
- **Backend:** Firebase
- Firebase Authentication
- Cloud Firestore
- Firebase Storage
- **Architecture:** Activity-based Android architecture
- **Version Control:** Git & GitHub

---

## 🌍 How It Works

1. User registers or logs in.
2. User submits a waste complaint with details and optional image.
3. Data is stored in Firebase Firestore.
4. Admin reviews the complaint.
5. Admin approves or rejects the report.
6. User can track the complaint status inside "My Reports".
